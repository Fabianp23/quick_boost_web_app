#QUICK BOOST WEB APP  
A Web application built on the Ruby On Rails framework.  
 * Ruby  
 * Ruby on Rails  
 * HTML  
 * CSS  
 * Sass  
 * JavaScript  
 * GEM: Paperclip  
 * GEM: Devise

##Features
1. Create Accounts  
2. Create Products  
 * Name  
 * Description  
 * Origin  
 * Picture (Upload)  
 * Delete a Product  
3. Comments  
 * Description  
 * Rating
